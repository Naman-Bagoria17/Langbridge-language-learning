# 🌍 LangBridge

A modern language learning platform that connects learners with native speakers, co-learners, and language teachers through real-time chat and video calls.

## ✨ Features

- 🤝 **Connect with Language Partners**: Find co-learners, native speakers, and teachers
- 💬 **Real-time Chat**: Instant messaging with clickable video call links
- 📹 **Video Calls**: High-quality video calling with Stream integration
- 👥 **Friend System**: Send and manage friend requests
- 🔔 **Notifications**: Stay updated with friend requests and messages
- 🎨 **Modern UI**: Beautiful, responsive design with DaisyUI themes
- 📱 **Mobile Friendly**: Fully responsive across all devices
- 🌲 **Forest Theme**: Default beautiful theme for new users
- 🔍 **User Search**: Find and connect with other language learners
- 🌐 **Language Selection**: Set your native and learning languages

## 🚀 Tech Stack

**Frontend:** React 18, Tailwind CSS, DaisyUI, React Query, Stream Chat & Video  
**Backend:** Node.js, Express.js, MongoDB, JWT Authentication, Stream API  
**Real-time:** Stream Chat SDK for messaging and video calls
